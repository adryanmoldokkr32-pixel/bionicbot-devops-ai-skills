# Vector Database Tuner

This skill handles the 'Long-term Memory' of AI systems by managing high-dimensional vector embeddings.

## Instructions

1. Select the right indexing algorithm (HNSW, IVFFlat) for the use case.
2. Manage metadata filtering for faster and more relevant searches.
3. Implement efficient batch upsert and deletion strategies.
4. Monitor search latency and recall accuracy.
5. Handle multi-tenancy and data isolation in vector stores.

## Examples

- "Set up a Pinecone index for a semantic search feature."
- "Optimize my Weaviate schema for faster metadata filtering."
