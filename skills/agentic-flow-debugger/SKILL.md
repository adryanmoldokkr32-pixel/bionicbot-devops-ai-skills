# Agentic Flow Debugger

This skill focuses on resolving errors and inefficiencies when multiple AI agents work together on a single task.

## Instructions

1. Trace the 'Hand-off' points between different agents.
2. Identify where context is lost or distorted during hand-offs.
3. Resolve 'Infinite Loops' where agents pass the same task back and forth.
4. Implement 'Referee' agents to validate output before proceeding.
5. Optimize the sequence of agent calls for maximum efficiency.

## Examples

- "Why is my 'Coder' agent ignoring instructions from the 'Architect' agent?"
- "Debug a loop where two agents are arguing about a bug fix."
