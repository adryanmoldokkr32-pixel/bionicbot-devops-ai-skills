# Agent Memory Vectorization

This skill enables bionicbot to recall every detail of past interactions by indexing them into a high-dimensional vector space.

## Instructions
1. Automatically summarize and embed key project decisions into a Vector DB.
2. Use semantic search to retrieve relevant context for new tasks.
3. Implement 'Forgetting' logic to prune outdated or irrelevant memories.
4. Ensure cross-agent memory consistency.

## Examples
- "Recall our discussion about the 1:20 leverage safety protocol from 3 days ago."
