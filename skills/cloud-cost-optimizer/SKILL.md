# Cloud Cost Optimizer

This skill analyzes cloud usage and identifies opportunities to save money on AWS, Azure, or Google Cloud.

## Instructions

1. Audit currently running resources and find underutilized instances.
2. Recommend Reserved Instances or Savings Plans for predictable workloads.
3. Suggest Spot Instances for non-critical, interruptible tasks.
4. Implement auto-scaling to match resources with actual demand.
5. Set up billing alerts and budgets to prevent surprise costs.

## Examples

- "Analyze my AWS bill and suggest 3 ways to reduce costs by 20%."
- "Find all unused S3 buckets and EBS volumes in my account."
