# Infrastructure as Code (Terraform)

This skill focuses on defining and provisioning cloud infrastructure using declarative configuration files.

## Instructions

1. Write Terraform configuration files (`.tf`) for cloud resources.
2. Manage state files securely (e.g., S3 backend with locking).
3. Implement modules for reusable infrastructure components.
4. Use `terraform plan` to preview changes before applying.
5. Handle resource dependencies and multi-cloud setups.

## Examples

- "Provision an AWS RDS database and an S3 bucket using Terraform."
- "Create a reusable Terraform module for a standard VPC setup."
