# SSL & Security Certificate Manager

This skill focuses on the core security of web traffic, managing certificates and encryption protocols.

## Instructions

1. Issue and renew SSL certificates (Let's Encrypt, Cloudflare).
2. Configure web servers (Nginx, Apache) for secure HTTPS.
3. Implement HSTS (HTTP Strict Transport Security) to force secure links.
4. Audit for weak ciphers or outdated TLS versions.
5. Monitor certificate expiration dates to prevent downtime.

## Examples

- "Set up a Let's Encrypt certificate for my new domain."
- "Check if my site is vulnerable to old TLS exploits."
