# Data Cleaning Specialist

This skill enables bionicbot to take raw, unorganized data and turn it into a high-quality, usable asset.

## Instructions

1. Identify and handle missing values (Imputation vs Deletion).
2. Detect and remove duplicates or outliers.
3. Standardize formats (Dates, Currencies, Phone numbers).
4. Normalize text data (Lowercasing, removing special chars).
5. Verify data integrity and types before output.

## Examples

- "Clean this CSV file and make sure all dates are in ISO format."
- "Prepare this raw web-scraped data for a machine learning model."
