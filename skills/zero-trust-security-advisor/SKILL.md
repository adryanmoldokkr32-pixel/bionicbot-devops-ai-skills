# Zero-Trust Security Advisor

This skill focuses on the principle of 'Never Trust, Always Verify' for all system access.

## Instructions

1. Implement Multi-Factor Authentication (MFA) for all entry points.
2. Use Identity-Aware Proxies (IAP) to protect private tools.
3. Segment networks to prevent 'Lateral Movement' during an attack.
4. Enforce 'Least Privilege' access for all API keys and users.
5. Log and audit every single access request to the system.

## Examples

- "Implement a zero-trust architecture for my internal company wiki."
- "Review my API key permissions to ensure they follow 'Least Privilege'."
